TURCOSA - Manual Documentation¶

Thank you for using TURCOSA and welcome to manual documentation. If you have any questions regarding usage of the software, please contact us via email: support@turcosasolutions.com.

About TURCOSA¶

TURCOSA is a cloud-based statistical analysis system that provides a simplified, easy-to-use interface and customizable options with comprehensive statistical analysis methods.

Simple interface: Simplify data analysis, get rid off all the unnecessary work, focus on your data.

Powerful analysis tool: Select more than 50 statistical analysis modules.

Customizable: Your data, your plan. Choose analysis modules which fits you and your data.

Beautiful visualization: Create interactive graphs to summarize your data.
